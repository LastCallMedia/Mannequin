Get Started

1.

Choose the extension you would like to use and install it.

While we would eventually like to provide support for all major frameworks and CMS systems, these are the extensions that are currently available.

HTML Extension

Display static HTML files as Mannequin patterns.
Twig Extension

Display Twig templates as Mannequin patterns.
Drupal Extension

Display Drupal 8 Twig templates as Mannequin patterns.

Configure

3.

Start a live development server using vendor/bin/mannequin server.
